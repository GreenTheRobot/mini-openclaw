基础运行架构

  • CLI：任务 / 图片输入
  • Agent Core：决策并调用工具
  • LLM Backend：DeepSeek / Qwen
  • Tool Layer：执行真实动作
  • Observation：结果回传与回答

任务规划与协作

├─复杂任务自动拆分 TODO
├─TODO 状态推进：pending → in_progress → completed
├─按任务需要启用角色分工
├─Research Agent：文献、网页、论文证据
├─Engineering Agent：代码、修改、测试、实验
├─Multimodal Agent：图片、图表、报错截图
└─Reviewer：检查证据、成功声明与风险
└─主 Agent：汇总多角色结果，生成最终回答

代码理解、修改与调试

  • 代码库探索：glob / grep / read
  • 最小化修改：read → edit → read 验证
  • Shell 执行：运行脚本、测试与复现命令
  • 工具失败恢复：读取错误 → 调整参数/换工具 → 重试
  • 截图调试：从终端或 IDE 报错图提取线索，再本地验证
